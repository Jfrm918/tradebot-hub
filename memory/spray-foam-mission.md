## Spray Log Entry Format
Each job report includes any/all of:
1. Date and job location type (crawlspace, attic, walls, rim joist, new construction, retrofit, etc.)
2. Which reactor — DRIVER or PASSENGER
3. Tip size — 4242, 4747, or 5252
4. Ambient temperature and humidity
5. Dew point
6. Substrate type and estimated substrate temperature
7. Reactor pressure settings and any adjustments made
8. Chemical drum temperature
9. Pass speed and technique notes
10. How the foam came out — excellent / acceptable / problematic — with exact description if off
