# FoamDial Brand Guide

## Inspiration
**Insulation Services of Tulsa (istulsa.com)**
- Professional insulation company in Tulsa, OK
- Jason's employer
- Established, trusted, local brand
- Focus on quality, efficiency, expertise

## Color Palette

### Primary Colors
- **Professional Blue:** `#0052CC` or `#003D99` (trust, reliability, technical expertise)
- **Safety Orange/Gold:** `#FF8C00` or `#FFA500` (energy, precision, visibility)
- **Charcoal Gray:** `#2C3E50` (professionalism, stability)

### Secondary Colors
- **Light Gray:** `#ECF0F1` (clean backgrounds)
- **Dark Gray:** `#34495E` (text, secondary elements)
- **Accent White:** `#FFFFFF` (clarity)

## Typography
- **Headers:** Sans-serif (bold, professional) — similar to istulsa.com
- **Body:** Clean, readable sans-serif
- **Accent:** Monospace for technical data/specs

## Visual Style
- Professional but approachable
- Technical credibility (charts, data visualization)
- Blue + Orange combination = trust + energy/precision
- Dark backgrounds with clean typography (like modern SaaS)

## Application Areas

### FoamDial Web Interface
- Blue as primary (buttons, headers, active states)
- Orange accents (alerts, performance indicators, CTA buttons)
- Charcoal/gray for backgrounds
- White text on dark for readability

### Mobile App
- Blue navigation, orange quick-action buttons
- Card-based layout (clean, scannable)
- Real-time job data in orange highlights
- Status indicators: Green (optimal), Orange (adjust), Red (critical)

### Dashboard/Analytics
- Blue line charts for performance trends
- Orange bars for achievement metrics
- Professional grid layout
- High contrast for field visibility (outdoors on jobsites)

## Logo Concept
- **FoamDial** wordmark
- Incorporate: dial/gauge imagery (precision), foam pattern (texture), blue + orange
- Professional, tech-forward aesthetic

## Brand Voice
- **Tone:** Professional, precise, helpful
- **Message:** "Precision spray foam dialing. Real-time intelligence. Consistent results."
- **Audience:** Spray foam installers (practical, results-focused)

## Use Cases
1. **Jobsite Dashboard:** Real-time temperature, humidity, dew point, recommended settings
2. **Mobile Alert:** "Optimal spray conditions in 2 hours"
3. **Historical Log:** Blue chart showing successful vs. problematic spray parameters
4. **Equipment Integration:** Orange warnings when conditions drift from optimal

## Rationale
- **Blue:** Matches professional construction/insulation industry standards (istulsa.com inspiration)
- **Orange:** High visibility on jobsite, signals precision + energy
- **Modern SaaS feel:** Appealing to forward-thinking installers who want technology advantage
- **Contractor-friendly:** Rugged, readable, not overcomplicated
